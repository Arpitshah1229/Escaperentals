# EscapeRentals

A full-stack Airbnb-style rental platform.

## Tech Stack

Frontend
- React
- Axios
- CSS

Backend
- Django
- Django REST Framework
- JWT Authentication

Database
- SQLite

## Features

- Property listings
- Booking system
- Reviews
- Admin dashboard
- Payments tracking
